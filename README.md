# Task Manager

## Installation

Clone repository

```bash
$ git clone https://github.com/iamRajatSharma/assignment-1
```

Change directory

```bash
$ cd assignment-1
```

Install dependency

```bash
$ npm i
```

Run docker container to start mongodb database

```bash
$ docker-compose up -d
```

Start project

```bash
$ npm start
```
